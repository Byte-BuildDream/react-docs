# 匹配系统文档索引

## 概述

匹配系统是 Elo 匹配平台的核心功能模块，负责处理玩家的队伍匹配、队列管理和比赛确认等关键功能。本文档索引提供了匹配系统相关文档的快速导航。

## 文档目录

### 核心组件

- **[匹配池系统](./matching-pools.mdx)** - 详细的匹配池管理组件文档
  - DefaultPool (5v5 常规匹配)
  - DoublePool (2v2 匹配)
  - EloPool (5v5 ELO ban图匹配)

### 相关服务

- **[玩家服务](../player-services.mdx)** - 玩家管理和实时通信
- **[队伍服务](../team-services.mdx)** - 队伍管理和状态同步
- **[休闲中心服务](../leisure-center-service.mdx)** - 自定义房间管理

## 快速导航

### 匹配池类型

| 匹配池 | 队伍人数 | 匹配模式 | 特殊功能 |
|--------|----------|----------|----------|
| DefaultPool | 5人 | 常规匹配 | 队伍平衡 |
| DoublePool | 2人 | 快速匹配 | 简单匹配 |
| EloPool | 5人 | ELO匹配 | Ban图系统 |

### 核心功能

- **队伍匹配**: 基于技能等级的智能匹配
- **队列管理**: 等待队列和已匹配队列的管理
- **比赛确认**: 队员准备状态和比赛开始流程
- **地图禁用**: ELO模式下的地图选择系统

### 技术特性

- **实时通信**: 基于 SignalR 的实时消息传递
- **线程安全**: 使用锁机制保护共享资源
- **异步处理**: 全异步操作提高性能
- **错误处理**: 完善的异常处理和恢复机制
