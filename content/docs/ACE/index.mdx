## 🚀 快速开始

### 1. 系统概述
Elo 匹配系统是一个基于 WebSocket 的实时游戏匹配平台，支持：
- 实时玩家匹配
- 队伍管理和协作
- 自定义房间创建
- 库存和奖励系统
- 好友和社交功能

### 2. 技术架构
- **后端框架**: ASP.NET Core
- **实时通信**: SignalR
- **数据库**: Supabase (PostgreSQL)
- **ORM框架**: Postgrest
- **日志框架**: NLog
- **编程语言**: C#

### 3. 核心组件
- **Hub服务**: 处理实时通信
- **控制器**: 处理HTTP请求
- **数据模型**: 数据库表结构
- **匹配池**: 玩家匹配算法
- **事件系统**: 系统事件处理
