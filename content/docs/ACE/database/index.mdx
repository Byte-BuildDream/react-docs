## 快速开始

### 1. 了解数据模型
首先阅读 [数据库基础模型](./base-models.mdx) 文档，了解系统中所有数据表的结构和关系。

### 2. 学习操作方法
查看各种数据库操作的示例代码，包括查询、插入、更新和删除操作。

### 3. 掌握错误处理
学习如何处理数据库连接超时、查询失败等异常情况。

### 4. 优化性能
了解如何优化数据库查询性能，提高系统响应速度。

## 技术栈

- **数据库**: Supabase (PostgreSQL)
- **ORM框架**: Postgrest
- **编程语言**: C#
- **日志框架**: NLog
- **序列化**: Newtonsoft.Json
